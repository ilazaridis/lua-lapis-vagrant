# lapis-vagrant
A vagrant box for lapis web framework

# Install
- Clone the repository
- vagrant up
- sync folder lapis should have been already created and openresty is up and running on http://localhost:9090

# What is included
- Ubuntu Trusty 14.04
- Lua 5.1
- Lapis
- Openresty 1.9.3.2
- Luarocks
- Moonscript
